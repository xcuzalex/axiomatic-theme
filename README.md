# Axiomatic Theme

A premium, high-contrast VS Code theme series designed for clarity, semantic precision, and a sophisticated aesthetic. Inspired by the depth of Tokyo Night and the logic of modern development.

## âœ¨ Features

- **Semantic Precision**: Distinct colors for every code construct (Structs, Enums, Traits, Macros, etc.) to help you read code faster.
- **Anti-Fog Design**: Deep, crisp backgrounds combined with vibrant, high-contrast foregrounds for maximum clarity.
- **Premium Palette**: A carefully curated "Storm" and "Night" inspired color scheme that feels high-end and professional.
- **Eye-Friendly**: Optimized for long coding sessions with reduced blue light strain and balanced saturation.

## ðŸŽ¨ Variants

### Axiomatic Dark (Storm)
A sophisticated dark theme featuring a deep, textured blue-grey background (`#24283b`).
- **Background**: Deep Storm Blue for a professional, focused environment.
- **Syntax Highlights**:
  - **Functions**: Vibrant Teal-Green (`#73DACA`) for clear logic flow.
  - **Variables**: Warm Golden-Wheat (`#E0AF68`) for data visibility.
  - **Properties**: Fresh Grass-Green (`#9ECE6A`) for object members.
  - **Types/Structs**: Elegant Lavender (`#BB9AF7`).
- **Terminal**: High-contrast ANSI colors for a sharp command-line experience.

### Axiomatic Light
A clean, high-contrast light theme for bright environments, mirroring the semantic logic of the dark variant.
